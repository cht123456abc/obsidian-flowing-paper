# 沉思模式 - Obsidian 插件

## 核心理念

在传统编辑器中，我们的视线随着光标不断向下移动。而在沉思模式中，光标固定在一个位置，内容向下流动——就像打字机上的纸张。

这种设计让你的注意力始终聚焦在"当下"，减少视线追踪，更适合快速思维捕捉和意识流写作。

## 行为对比

### 传统编辑模式
```
光标在第1行 → 按回车 → 光标移到第2行
内容位置不变
```

### 沉思模式
```
光标在第1行 → 按回车 → 光标仍在第1行（行首）
当前行内容下沉到第2行
```

## 快速开始

1. 安装插件
2. 按 `Ctrl/Cmd + Shift + F` 开启沉思模式
3. 开始写作，体验不同的思维流动方式

## 设计哲学

> "让思绪沉淀，让当下永驻"

这个插件的设计灵感来自：
- 打字机的固定光标体验
- 禅宗的"活在当下"理念
- 意识流写作的需求
- 减少认知负担的人机交互设计
